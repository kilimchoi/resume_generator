HTML Resume Builder

This tool generates a basic HTML resume for Jobmine. In addition to the default basic info, you can add the follow blocks:

  • Generic Block: A block with sub-headings and descriptions.
  • History Block: Similar to a generic block but also includes start and end date fields.

Save and Generate only works if your browser supports HTML5 Local Storage. Everything is saved locally to your current machine and browser. If you try to use this tool on a different machine (another laptop or computer) or browser (Firefox or Chrome), you will not see your previously saved data. This tool autosaves every 10 seconds.
